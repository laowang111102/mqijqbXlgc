# Python在线考试系统 python666

## 项目概述

本项目是一个基于Python语言的在线考试系统，它集成了学院、班级、考试科目、学生、教师等多种信息管理功能，致力于为校园考试提供便捷的在线管理解决方案。

## 技术栈

- **前端**: Vue
- **后端**: Python + Django
- **数据库**: MySQL
- **开发工具**: PyCharm、Navicat
- **运行环境**: 
  - Python 3.0以上
  - MySQL 8.0

## 功能模块

- 学院信息管理
- 班级信息管理
- 考试科目管理
- 学生信息管理
- 教师信息管理
- 校园考试管理

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- Python环境（3.0版本以上）
- MySQL数据库（8.0版本）
